# Student Management System
A console application developed in Python to manage basic student information.
---
## Features

### Product Management (CRUD)
- Add Student
- Display Students
- Search Student
- Update Student
- Delete Student

### 💾 Data Persistence
- Save inventory to CSV files
- Load inventory from CSV files
- Data validation on upload

## Technologies
- Python 3

## Installation

Follow these steps to install and run the project on your computer:

### Clone the repository

```bash
git clone https://github.com/Isaac-G17/StudentManagementSystem.git
```
### Open the project folder
```bash
cd StudentManagementSystem
```
### Create a virtual environment

```bash
python3 -m venv .venv
```
### Activate the virtual environment
#### Linux / macOS
```bash
source venv/bin/activate
```
#### Windows
```bash
venv\Scripts\activate
```
### Run the program

```bash
python3 main.py
```
